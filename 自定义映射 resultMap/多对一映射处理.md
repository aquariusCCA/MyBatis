> 查询员工信息以及员工所对应的部门信息

```java
@Data  
@NoArgsConstructor  
@AllArgsConstructor  
public class Emp implements Serializable {  
    private Integer eid;      // PK，自增  
    private String empName;  // 對應 emp_name    private Integer age;  
    private Integer sex;      // 1=男, 2=女  
    private String  email;  
    private Integer did;  
    private Dept dept;  
}
```

```java
@Data  
@AllArgsConstructor  
@NoArgsConstructor  
public class Dept {  
    private Integer did;  
    private String deptName;  
}
```
