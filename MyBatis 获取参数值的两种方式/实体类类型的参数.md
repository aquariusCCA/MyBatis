> 若 mapper 接口中的方法参数为实体类对象时此时可以使用 `${}` 和 `#{}`，通过访问实体类对象中的属性名获取属性值，注意 `${}` 需要手动加单引号

```java
/**  
 * 對應表：t_user  
 * 注意：sex = 1(男), 2(女)  
 */@Data  
@NoArgsConstructor  
@AllArgsConstructor  
public class User implements Serializable {  
    private Integer eid;      // PK，自增  
    private String empName;  // 對應 emp_name    private String password;  
    private Integer age;  
    private Integer sex;      // 1=男, 2=女  
    private String  email;  
    private Integer did;  
}
```

```java
/**
 * 添加用户信息
 *
 * @param user
 * @return
 */
int insertUser(User user);
```

```xml
<!--int insertUser(User user);-->  
<insert id="insertUser">  
    insert into t_user values(null, #{empName},#{password},#{age},#{sex},#{email},null)  
</insert>
```

```java
@Test  
public void insertUser() {  
    SqlSession sqlSession = SqlSessionUtils.getSqlSession();  
    ParameterMapper mapper = sqlSession.getMapper(ParameterMapper.class);  
    User user = new User(null,"Tom","123456",12,1,"123@321.com", null);  
    mapper.insertUser(user);  
}
```
